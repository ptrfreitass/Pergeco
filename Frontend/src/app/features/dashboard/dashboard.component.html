🧪 Como Testar o Interceptor

  1. Abra o DevTools (F12)
  2. Vá para a aba Console
  3. Vá para a aba Network (Rede)
  4. Clique no botão "🧪 Testar Interceptor" acima
  5. No Console, você verá logs do interceptor
  6. Na aba Network, procure pela requisição me
  7. Clique nela e vá para Headers
  8. Procure por Authorization: Bearer {{ '{' }}token{{ '}' }} nos Request Headers
